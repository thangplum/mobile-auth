# skipli-assignment

## Description
This is the frontend and backend of Skipli's coding challenge

## Frontend
In the project directory, you can run:
### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
